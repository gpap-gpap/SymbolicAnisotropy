# Symbolic Anisotropy

A Wolfram Language package for symbolic calculations in anisotropic elasticity.

## Requirements

_Mathematica >13.1_

## Installation

`<<"https://github.com/gpap-gpap/SymbolicAnisotropy/SymbolicAnisotropy.wl"`
